About ProxMate

Our mission: Making proxies fun and a joy to use!

The idea

  • Imagine you could set up a proxy, with just a click. And when you don't need it anymore, it's gone.
  • Imagine you are in 10 countries at the same time. Over the same connection.
  • Imagine you can create your own config files and share them with the world.
  • Imagine you use a proxy, without any speed losses.
  • Imagine you start enjoying using proxies.
  • Imagine you can fly.

    • ...we want to give you all that - as easy as possible!

The tool

ProxMate is a proxy package manager designed to be as modular as possible. Each package is a proxy configuration and can be used independently or together with others.

Each package can have a different country and each package can do completely different things, all coming together in a central hub right in your browser!

Development

ProxMate is currently being maintained and developed by Personalitycores, a super small startup based in Germany. (Seriously, if we tell you how many people work here, you would laugh.)

Our baby started as a simple unblocker for Youtube but grew bigger than we could have ever expected. We even hit the 500.000 user mark, can you imagine? A HALF MILLION USERS!

There are some exciting stories I could tell you. For example how we managed to generate 120TB of traffic in 1 month! That made our hosters pretty angry, I can tell you!

Technology

ProxMate is put together by a wide range of different technologies. Everything we use is open and available for you to try out yourself!

  • Addon: coffeescript, javascript, angularjs, jasmine, mocha, chai (source)
  • API server: javascript, nodejs, mongodb (source)
  • This website: angularjs, foundation and... nothing. Just these 2 (source)
  • Accelerating and CDN: Kindly provided by fastly! They are awesome!
  • Static page hosting: amazon s3
  • Proxy software: Squid3
